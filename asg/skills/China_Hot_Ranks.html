# API 参考文档

## DailyHotAPI 类

### 初始化

```python
from hot_ranks import DailyHotAPI

api = DailyHotAPI(base_url="http://localhost:6688", timeout=30)
```

### get_hot_rank(platform, params=None)

获取指定平台的原始热榜数据。

```python
data = api.get_hot_rank("weibo")
```

返回值：
- 成功时返回 DailyHotApi 的 JSON 字典
- 失败时返回 `None`

---

## HotRanksAggregator 类

### 初始化

```python
from hot_ranks import HotRanksAggregator

aggregator = HotRanksAggregator(api_url="http://localhost:6688")
```

### 主要属性

- `PLATFORMS`: 所有支持平台的映射表
- `DEFAULT_PLATFORMS`: 默认抓取的 6 个平台
- `ALL_PLATFORMS`: 全量平台列表
- `results`: 最近一次抓取成功的平台结果

### fetch_platform(platform, keyword=None, verbose=True)

抓取单个平台热榜，并将成功结果写入 `results`。

```python
aggregator.fetch_platform("zhihu")
aggregator.fetch_platform("weibo", keyword="AI")
```

返回值：
- `True`: 抓取并筛选后有结果
- `False`: 平台不存在、接口失败或无匹配结果

### fetch_all(platforms=None, keyword=None)

抓取多个平台热榜。

```python
# 默认 6 个平台
results = aggregator.fetch_all()

# 指定平台
results = aggregator.fetch_all(["weibo", "zhihu", "bilibili"])

# 全量平台关键词筛选
results = aggregator.fetch_all(aggregator.ALL_PLATFORMS, keyword="AI")
```

返回值：
- 返回 `results` 字典，键为平台标识，值包含平台名、平台链接、热榜条目和更新时间

### list_platforms()

在终端打印所有支持的平台列表。

```python
aggregator.list_platforms()
```

### export_markdown(output_file=None)

将最近一次抓取结果导出为 Markdown。

```python
aggregator.fetch_all()
aggregator.export_markdown("hot-ranks.md")
```

### export_html(output_file=None)

将最近一次抓取结果导出为 HTML 页面。页面按平台分栏目展示，每个平台一张卡片。

```python
aggregator.fetch_all(aggregator.ALL_PLATFORMS, keyword="AI")
aggregator.export_html("hot-ranks-ai.html")
```

---

## 命令行参数

```bash
python3 hot_ranks.py [platform]
```

参数说明：
- 留空：抓取默认 6 个平台
- `all`: 抓取全部 36 个平台
- `list` 或 `--list`: 列出所有支持平台
- `weibo`、`zhihu`、`bilibili`: 抓取单个平台
- `weibo,zhihu,bilibili`: 抓取多个指定平台

可选参数：
- `--keyword`, `-k`: 关键词筛选
- `--api`: 自定义 DailyHotApi 地址
- `--output`, `-o`: 导出 Markdown 文件
- `--html-output`: 导出 HTML 页面

示例：

```bash
# 默认平台
python3 hot_ranks.py

# 全量平台
python3 hot_ranks.py all

# 关键词筛选
python3 hot_ranks.py all --keyword AI

# Markdown 导出
python3 hot_ranks.py --output hot-ranks.md

# HTML 导出
python3 hot_ranks.py all --html-output hot-ranks.html
```

import requests
import json
import sys
from datetime import datetime
from html import escape
from typing import List, Dict, Optional


class DailyHotAPI:
    """DailyHotApi 客户端"""
    
    def __init__(self, base_url: str = "http://localhost:6688", timeout: int = 30):
        self.base_url = base_url
        self.timeout = timeout
        self.session = requests.Session()
        self.session.headers.update({
            'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36'
        })
    
    def get_hot_rank(self, platform: str, params: Optional[Dict[str, str]] = None) -> Optional[Dict]:
        """获取指定平台热榜"""
        try:
            url = f"{self.base_url}/{platform}"
            response = self.session.get(url, params=params, timeout=self.timeout)
            response.raise_for_status()
            data = response.json()
            
            if data.get('code') == 200:
                return data
            else:
                print(f"⚠️  {platform} 返回错误：{data.get('message', 'Unknown error')}")
                return None
        except requests.exceptions.Timeout:
            print(f"⚠️  {platform} 请求超时 ({self.timeout}秒)")
            return None
        except requests.exceptions.ConnectionError:
            print(f"⚠️  无法连接到 DailyHotApi 服务 ({self.base_url})")
            return None
        except Exception as e:
            print(f"⚠️  {platform} 获取失败：{e}")
            return None

# 🔥 中国热榜聚合器 - China Hot Ranks Aggregator

基于 [DailyHotApi](https://github.com/imsyy/DailyHotApi) 的热榜聚合工具，支持 **36 个平台** 的热榜抓取。

## 🎯 特性

- ✅ **36 个平台**：微博、知乎、B 站、抖音、CSDN、掘金等
- ✅ **稳定可靠**：本地服务调用，无网络限流问题
- ✅ **快速高效**：单次抓取 <10 秒
- ✅ **易于扩展**：新增平台只需一行配置
- ✅ **多种输出**：终端显示、Markdown 导出、HTML 页面导出

# 中国热榜聚合器 📊

> **一键获取全网热点，掌握流量密码**

支持 7 大主流平台的热门内容获取：微博、B 站、百度、CSDN、GitHub、知乎、抖音。

---

## ✨ 特性

- 🎯 **7 大平台覆盖** - 社交媒体、视频、搜索、技术社区全覆盖
- 🚀 **一键获取** - 单命令获取所有平台热榜
- 📝 **格式化输出** - 带链接、热度指数、播放量等详细信息
- 🔧 **灵活选择** - 支持获取全部或指定平台
- 📦 **开箱即用** - 无需 API Key，基于现有工具链
